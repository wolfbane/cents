Manage investment theses.

## Synopsis

```bash
cents thesis <subcommand> [OPTIONS] [ARGS]...
```

## Subcommands

- `cents thesis close` — Close a thesis with outcome assessment.
- `cents thesis create` — Create a new investment thesis.
- `cents thesis list` — List theses.
- `cents thesis show` — Show thesis details.
- `cents thesis twin` — Create a policy-neutral paired twin of an existing directional thesis.
- `cents thesis update` — Update a thesis.

### `cents thesis close`

Close a thesis with outcome assessment.

**Synopsis**

```bash
cents thesis close [OPTIONS] THESIS_ID
```

**Arguments**

| Argument | Type | Required |
| --- | --- | --- |
| `THESIS_ID` | `text` | yes |

**Options**

| Option | Type | Default | Description |
| --- | --- | --- | --- |
| `--outcome/-o [correct|incorrect|partial|unclear]` | `[correct \| incorrect \| partial \| unclear]` |  | Was the thesis correct? |
| `--notes/-n TEXT` | `text` |  | Closing notes (updates hypothesis) |

**Example**

```bash
cents thesis close [OPTIONS] THESIS_ID
```

### `cents thesis create`

Create a new investment thesis.

    Example: cents thesis create --title "NVDA bull case" --symbol NVDA

**Synopsis**

```bash
cents thesis create [OPTIONS]
```

**Options**

| Option | Type | Default | Description |
| --- | --- | --- | --- |
| `--title/-T TEXT` | `text` |  | Thesis title |
| `--hypothesis/-h TEXT` | `text` |  | Detailed thesis statement |
| `--tags/-t TEXT` | `text` |  | Comma-separated tags |
| `--symbol/-S TEXT` | `text` |  | Stock ticker (e.g., AAPL) |
| `--business-quality/-b TEXT` | `text` |  | Quality assessment |
| `--valuation/-v [undervalued|fair|overvalued]` | `[undervalued \| fair \| overvalued]` |  | Valuation assessment |
| `--moat/-m TEXT` | `text` |  | Competitive advantage |
| `--time-horizon/-H [short|medium|long]` | `[short \| medium \| long]` |  | Investment horizon |
| `--horizon-end TEXT` | `text` |  | Expiry date (YYYY-MM-DD) |
| `--risks/-r TEXT` | `text` |  | Comma-separated key risks |
| `--target-price FLOAT` | `float` |  | Target price to close thesis |
| `--stop-price FLOAT` | `float` |  | Stop price to close thesis |
| `--cohort [directional|neutral]` | `[directional \| neutral]` | `directional` | Cohort: directional (default) or neutral (paired hedge) |
| `--hedge-with TEXT` | `text` |  | Hedge symbol for neutral cohort theses |
| `--from-research TEXT` | `text` |  | Auto-populate from research on symbol |

**Example**

```bash
cents thesis create --title "NVDA bull case" --symbol NVDA
```

### `cents thesis twin`

Create a policy-neutral paired twin of an existing directional thesis.

    Example: cents thesis twin abc123 --hedge-with SPY

**Synopsis**

```bash
cents thesis twin [OPTIONS] THESIS_ID
```

**Arguments**

| Argument | Type | Required |
| --- | --- | --- |
| `THESIS_ID` | `text` | yes |

**Options**

| Option | Type | Default | Description |
| --- | --- | --- | --- |
| `--hedge-with TEXT` | `text` |  | Symbol to short against the parent's long leg |

**Example**

```bash
cents thesis twin abc123 --hedge-with SPY
```
