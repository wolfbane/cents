Cents exposes 22 top-level command groups, each grouping related operations
on theses, research, watchlists, alerts, positions, broker state, the
autonomous factory loop, pre-registered experiments, calibration, and the
LLM eval harness.

The pages below are auto-generated from the CLI's `--help` output, so they
stay in sync with the installed `cents` binary. Each page lists subcommands,
options, types, defaults, and a usage example.

:::note[Generator status]
Per-command pages are populated by `scripts/generate_docs.py` (Phase 2 of
the launch plan). Until that generator runs, the cards below link to pages
that may not yet exist — start with the [Quickstart](/quickstart/) for
runnable examples.
:::

## How regeneration works

The generator walks the Click command tree (`cents.cli.cli.commands`) and
emits one MDX file per group into this directory. Hand-written prose lives
in sibling `*.intro.mdx` files that get prepended at generation time, so
regenerating after a CLI change never clobbers the narrative.
