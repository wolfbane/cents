<Aside type="caution" title="Research tool, not an investing tool">
Cents is an open-source experiment in agent-orchestrated investment
research. It is **not** an investment adviser, broker, or
recommendation engine, and nothing it emits is investment advice.
Real-money trading is technically possible but **explicitly out of
scope** — the autonomous loop is hard-coded to paper, and the
controls a real trading system would require (vol-scaled sizing,
drawdown gates, slippage and borrow modeling, reconciliation, audit
recordkeeping) are intentionally absent. [Read the full scope
statement](/scope/) before going further.
</Aside>

Cents is a CLI sandbox for treating investing as a hypothesis-testing
exercise — write down a thesis, declare which regime variables it
depends on, and let the system flag when one of those dependencies
shifts. Pluggable screeners discover candidate universes, specialised
research agents gather evidence, and an autonomous factory loop walks
the cycle in **paper mode only** — producing a labeled outcomes
dataset you can study.

```bash
git clone https://github.com/wolfbane/cents.git
cd cents && pip install .
```

<CardGrid>
  <Card title="Thesis-driven" icon="pencil">
    Every position starts as a written hypothesis with a valuation,
    horizon, target, and stop. Research is evaluated _against the thesis_,
    not in the abstract.
  </Card>
  <Card title="Regime-aware" icon="seti:notebook">
    Each thesis declares the policy / macro variables it depends on
    (`tariffs.china`, `fed_policy`, `ai_capex`…). An EventAgent ingests
    Federal Register policy events; when an event hits one of your tags,
    you get a **premise-invalidation alert** — before the market fully
    prices it.
  </Card>
  <Card title="Multi-agent" icon="puzzle">
    Eight specialised agents — fundamentals, technical, macro, sentiment,
    moat, insider, event, plus an orchestrator — pull from FMP, Alpaca,
    FRED, NewsAPI, and the Federal Register; each contributes evidence
    and a conviction delta.
  </Card>
  <Card title="Paper-only factory" icon="setting">
    `cents factory run` walks a universe of symbols and opens
    **paper theses** where the orchestrator clears your entry
    threshold (with neutral twins in paired mode), closing on target /
    stop / horizon / premise-invalidation. Hard-coded to paper — real
    trading is [out of scope](/scope/).
  </Card>
  <Card title="Pluggable screeners" icon="seti:search">
    Five built-in discovery strategies — value, growth, momentum,
    mean-reversion, insider-cluster — that turn a parent universe into
    a candidate list the factory walks. Every thesis records which
    screener produced it, so `cents factory analyze --by discovery`
    surfaces which discovery strategy wins in which regime.
  </Card>
  <Card title="Accuracy-tracking" icon="approve-check">
    Outcomes are recorded with a closure _reason_ —
    correct / incorrect / unclear / **invalidated** / **preempted** —
    plus the regime snapshot captured at thesis birth, so you can
    stratify later instead of averaging across regimes that have nothing
    to do with each other.
  </Card>
  <Card title="Cost-visible" icon="seti:db">
    Every Anthropic call is recorded to an `llm_usage` table; `cents
    usage summary` reports input/output/cache tokens and estimated cost
    by agent, model, day, or operation. Nothing about the cost of
    autonomy is hidden.
  </Card>
</CardGrid>
