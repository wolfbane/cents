# Examples

This directory holds sample exports used by the docs site.

- `research-NVDA.html` — generated via `cents research NVDA --export-html examples/research-NVDA.html` (requires API keys configured).
- `demo.cast` — asciinema recording of an end-to-end session (`asciinema rec examples/demo.cast`).

Both are referenced by the website at `/Users/matthew/Projects/cents/website/` and should be regenerated whenever the CLI surface changes meaningfully.
