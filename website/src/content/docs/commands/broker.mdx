<Aside type="caution" title="Paper trading only">
The broker commands are wired to Alpaca's **paper-trading** endpoints.
Real-money trading is technically possible (Alpaca supports it) but
[explicitly out of scope](/scope/) for this project — none of the
controls a live trading system would require (vol-scaled sizing,
drawdown gates, slippage and borrow modeling, broker reconciliation,
lot accounting) exist. Do not point these commands at a live account.
</Aside>

Alpaca broker integration (paper-trading only).

## Synopsis

```bash
cents broker <subcommand> [OPTIONS] [ARGS]...
```

## Subcommands

- `cents broker buy` — Buy shares (paper trading).
- `cents broker list` — List positions from broker.
- `cents broker sell` — Sell shares (paper trading).
- `cents broker status` — Check broker connection and account status.
- `cents broker sync` — Sync positions from broker to cents.

### `cents broker buy`

Buy shares (paper trading).

    Example: cents broker buy NVDA --qty 10

**Synopsis**

```bash
cents broker buy [OPTIONS] SYMBOL
```

**Arguments**

| Argument | Type | Required |
| --- | --- | --- |
| `SYMBOL` | `text` | yes |

**Options**

| Option | Type | Default | Description |
| --- | --- | --- | --- |
| `--qty/-q FLOAT` | `float` |  | Number of shares |
| `--thesis/-t TEXT` | `text` |  | Link to thesis ID |
| `--yes` | `boolean` | `false` | Confirm the action without prompting. |

**Example**

```bash
cents broker buy NVDA --qty 10
```
