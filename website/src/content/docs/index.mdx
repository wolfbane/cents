Cents is a CLI for investors who want to treat investing like a
hypothesis-testing exercise — write down a thesis, declare which regime
variables it depends on, and let the system warn you the moment one of
those dependencies shifts. Eight specialised research agents gather the
evidence; an autonomous factory loop can run the whole cycle for you and
produce a labeled outcomes dataset you can actually learn from.

```bash
git clone https://github.com/wolfbane/cents.git
cd cents && pip install .
```

<CardGrid>
  <Card title="Thesis-driven" icon="pencil">
    Every position starts as a written hypothesis with a valuation,
    horizon, target, and stop. Research is evaluated _against the thesis_,
    not in the abstract.
  </Card>
  <Card title="Regime-aware" icon="seti:notebook">
    Each thesis declares the policy / macro variables it depends on
    (`tariffs.china`, `fed_policy`, `ai_capex`…). An EventAgent ingests
    Federal Register policy events; when an event hits one of your tags,
    you get a **premise-invalidation alert** — before the market fully
    prices it.
  </Card>
  <Card title="Multi-agent" icon="puzzle">
    Eight specialised agents — fundamentals, technical, macro, sentiment,
    moat, insider, event, plus an orchestrator — pull from FMP, Alpaca,
    FRED, NewsAPI, and the Federal Register; each contributes evidence
    and a conviction delta.
  </Card>
  <Card title="Autonomous factory" icon="setting">
    `cents factory run` walks a universe of symbols, opens paper theses
    where the orchestrator clears your entry threshold (with neutral
    twins in paired mode), closes positions on target / stop / horizon /
    premise-invalidation, and writes a structured run log with token
    cost per run.
  </Card>
  <Card title="Accuracy-tracking" icon="approve-check">
    Outcomes are recorded with a closure _reason_ —
    correct / incorrect / unclear / **invalidated** / **preempted** —
    plus the regime snapshot captured at thesis birth, so you can
    stratify later instead of averaging across regimes that have nothing
    to do with each other.
  </Card>
  <Card title="Cost-visible" icon="seti:db">
    Every Anthropic call is recorded to an `llm_usage` table; `cents
    usage summary` reports input/output/cache tokens and estimated cost
    by agent, model, day, or operation. Nothing about the cost of
    autonomy is hidden.
  </Card>
</CardGrid>
