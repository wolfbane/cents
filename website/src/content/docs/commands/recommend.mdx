<Aside type="caution" title="Model signal, not investment advice">
The output of `cents recommend` is a **model signal**, not a recommendation
to buy or sell. The CLI does not perform suitability checks, factor risk
analysis, or position sizing. See [/scope/](/scope/) for the project's
stated boundaries.
</Aside>

Emit model signals (bullish / bearish / neutral) for open theses.

    Evaluates each open thesis against current prices, conviction, and
    target/stop/expiry to produce a *model signal*. Signals are not
    investment advice — see /scope/ for the project's stated boundaries.

    Rules (in priority order):
      1. Stop loss triggered          → CLOSE
      2. Target price reached         → CLOSE
      3. Thesis invalidated           → CLOSE / REVIEW
      4. Thesis expired               → REVIEW
      5. Low conviction w/ position   → bearish signal
      6. High conviction, no position → bullish signal
      7. Otherwise                    → neutral signal

## Synopsis

```bash
cents recommend [OPTIONS]
```

## Options

| Option | Type | Default | Description |
| --- | --- | --- | --- |
| `--buy-threshold FLOAT` | `float` | `70.0` | Minimum conviction to emit a bullish signal (default: 70) |
| `--sell-threshold FLOAT` | `float` | `30.0` | Maximum conviction to emit a bearish signal (default: 30) |
| `--output [text|json]` | `[text \| json]` |  | Output format (default: from config) |
| `--actionable` | `boolean` | `false` | Only show actionable signals (exclude neutral) |

## Example

```bash
cents recommend [OPTIONS]
```
